6 Create Super Admin User
After running the schema, create your admin user:
  1. Go to Supabase Dashboard → Authentication → Users
  2. Click "Add user" → "Create new user"
  3. Enter your admin email + password, check "Auto Confirm User"
  4. Copy the generated UUID from the users list
  5. Run in SQL Editor:
INSERT INTO users (id, email, role) VALUES ('paste-your-uuid-here', 'admin@yourdomain.com', 'super_admin');
7 Create Storage Buckets
  1. Go to Supabase → Storage → "New Bucket"
  2. Name: menu-images — check ✓ Public bucket → Create
  3. Name: menu-models — check ✓ Public bucket → Create
8 Deploy to Vercel via GitHub
  1. Push the project to your GitHub: github.com/Tarit23
  2. Go to Vercel Dashboard → "Add New Project"
  3. Import the GitHub repo
  4. Framework: Other, Root Directory: /
  5. Deploy → copy the Vercel URL
  6. Update APP_CONFIG.domain in js/supabase-config.js with the Vercel URL
  7. Update Razorpay webhook URL to the Supabase Edge Function URL
